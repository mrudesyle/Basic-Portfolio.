About Me

Hi there. I'm Mark Rudesyle. I currently work at KP Kauffman Co as a Database Engineer. I have a BM in Music Performance from Metro State Univ of Denver and a Masters of Science in Database Technologies from Regis University. Some of my daily activities include:

Database design, Database performance tuning, Defining database tables, indexes, views, constraints, triggers, stored procedures; Pro-actively managing and monitoring the production database environment; Responsible for import/export of data into databases; Responsible for creating, managing and implementing SQL Integration Services Packages; Responsible for creating, managing and implementing SQL Reporting Services Reports.
